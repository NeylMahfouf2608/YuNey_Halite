This uses the included Makefile to build the sample bot and starter kit.

Linux users:\+ if you\textquotesingle{}re having problems running the bundled {\ttfamily halite} executable, you will probably want to build your own from source. Grab the source download from the Downloads page, extract it, and run {\ttfamily make} inside the extracted directory. Then, move the built {\ttfamily halite} executable to this directory. You will need GCC 7.\+2 or newer, but preferably the latest version of GCC.
